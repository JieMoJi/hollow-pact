# AI-Gesture-Game-
A gesture-controlled game using MediaPipe hand tracking. Players use fist (charge) and palm (blast) to interact.
# AI Gesture Game

## 安装依赖
pip install mediapipe opencv-python numpy

## 运行手势测试
python test.py

## 文件结构
- gesture_module.py : 手势识别模块，提供 GestureRecognizer 类（Jie）
- game_engine : 一个简单引擎（Jie）
- test_integration : 测试脚本（Jie）
